# nest-starter

This repository is nestjs starter **without cli**

## How to use?

```bash
git clone https://github.com/BBackjk/nest-starter.git project
cd project
npm install
```

## Script

### Build

```bash
npm run build
```

### Start product

```bash
npm start
```

### Start development

```bash
npm run start:dev
```
